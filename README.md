# ESP8266-Web-Alarm-Clock-with-Custom-Greetings
An open-source ESP8266-based smart alarm clock featuring a MAX7219 LED matrix display, DS1307 RTC, NTP synchronization, Wi-Fi Access Point with responsive web configuration, EEPROM-based settings storage, multiple alarms, custom greeting messages, hourly chime, date/day display, and physical alarm stop button.
